# DockWP
## Docker + Nginx + Wordpress + Redis

### Feature
- Custom wordpress cron job
- Support Redis cache
- Fast setup with docker-compose 

### Automated build on Dockerhub
https://hub.docker.com/r/ph03n1x/dockwp/

### Project requirement
- docker
- docker-compose

### Build image
`docker-compose build`

### Test build
`docker-compose up`
